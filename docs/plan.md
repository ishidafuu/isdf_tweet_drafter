# Voice2Tweet Bot - タスク管理

## Phase 1: 環境構築

### 概要
プロジェクトの基盤となるファイルとディレクトリ構造を整備し、必要な設定ファイルを作成する。

### タスク

#### 🔴 高優先度
- [x] プロジェクト管理構造の導入（.claude/commands, docs/）
- [ ] `.env.example` ファイルを作成
- [ ] `.gitignore` に `.env` を追加済みか確認
- [ ] `requirements.txt` の内容確認・更新

#### 🟡 中優先度
- [ ] README.md の更新（claude-commands-template の使い方を追加）
